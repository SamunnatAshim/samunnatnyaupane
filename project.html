Web Graphics

Web graphic refers to visual images using on a Web site.The images in the web site is use to enhance or enable the representation
of idea or feeling, in order to reach the web site user.

Web or internet system is the technicwhere text, picture, video, audio are transmitted from one compute to anther computer.
If the sizes of these materials are big then it takes a lot of time to upload and down load. Whe have to such a file which
has small file size. There are many file formates but only some are important for web.

Among these file PNG file format is better for Web because PNG format is a lossless compression file format, which makes
it a common choice for user on the Web. PNG is a good choice for storing line drawings, text, and iconic graphics at small file size.
Jpg format is a lossy compressed file format.
