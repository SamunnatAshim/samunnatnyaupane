Effective use of web graphics

  1. Graphics is the first attraction of the webpage. Grapics should be presented with the purpose,organization and style of the page.

  2. Graphics should increase the disgn,structure or inforamtive content of the webpage without confusionor distracting the attention.

  3. Be conform weather the graphic fit on the webpage or not.

  4. Do not use big file size which take a lot time to open the page.

  5. Do not use strong graphic element which can pull attention away from central content or compete with another on the page.
    Because graphic should help to guide the viewer's focus to the important content on the page.

  6. Do not use repetitive,hateful,and horrible images in the webpage.

  7. Do not use graphics to express textual information .Something it may be irriting.

  8. Use textual equivalent alternatives for graphic content.

  9. Be sure there is suficient contrast between the text and the baground color when using text in graphics.

  10. Design graphic elements so that users can easily see the text from it's background.

  11. Be sure to avoid color combinations [RGB] that color blind users will not be to distinguish apart from one another.

  12. Avoid the use of graphics to convey texual content information.

  13. Always remember that using visually strong graphic elements on a page can be useful in directing viewers' attention and providing structure of the page.

  14. Bright and eye catching images may be attractive at first, but after some time you stop feeling interested or excited about the image because it is no longer new, it was funny for a while but the novelty soon wore off. They may cause viewers to lose patience with the site.
